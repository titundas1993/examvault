## Build Instructions

### Prerequisites
1. Android Studio Hedgehog (2023.1.1) or newer
2. JDK 17 (Android Studio will install it for you)
3. Android SDK Platform 34 (Android Studio will install it)

### Step 1: Add Keystore (CRITICAL)

The original keystore `examvault-release.keystore` is in:
`/home/z/my-project/examvault/play-store-final/examvault-release.keystore`

Copy it to:
`app/examvault-release.keystore`

If you don't have the keystore:
- For Play Store update: You MUST use the original keystore (or Play App Signing)
- For fresh test build: Just build debug APK (no keystore needed)

### Step 2: Open Project in Android Studio

1. Open Android Studio
2. File → Open → Select this folder (`examvault-android`)
3. Wait for Gradle sync to finish (downloads dependencies — first time takes 5-10 minutes)

### Step 3: Test Build (Debug APK)

Build → Build Bundle(s) / APK(s) → Build APK(s)

Output: `app/build/outputs/apk/debug/app-debug.apk`

Install on phone:
```
adb install app/build/outputs/apk/debug/app-debug.apk
```

### Step 4: Release Build (AAB for Play Store)

Build → Generate Signed Bundle / APK → Android App Bundle →
  - Keystore: app/examvault-release.keystore
  - Keystore password: ExamVault2026!
  - Key alias: examvault
  - Key password: ExamVault2026!

Output: `app/build/outputs/bundle/release/app-release.aab`

## AdMob Test Notes

After installing on a real device:

1. First ad loads 10-15 seconds after app starts
2. Ad triggers when:
   - User opens a category (Home → SSC etc.)
   - User taps a banner with click action
   - User submits a test (best trigger)
3. Minimum 90 seconds between ads
4. Premium users see NO ads (auto-skipped)
5. Check ads status via logcat:
   ```
   adb logcat -s ExamVault
   ```

## Troubleshooting

**Gradle sync fails?**
- Check internet connection
- Try File → Sync Project with Gradle Files
- Delete `.gradle` folder and sync again

**Build fails with "SDK not found"?**
- Open SDK Manager → install Android SDK 34
- Open Project Structure → set SDK location

**App crashes on launch?**
- Make sure phone has internet
- Check logcat: `adb logcat -s ExamVault`
- Verify App URL is reachable in browser

**AdMob ad not showing?**
- Add your test device ID to `MainActivity.java`:
  Replace `"YOUR_TEST_DEVICE_ID"` with actual device ID from logcat
- Real ads only show after AdMob account approval (post Play Store publish)
- For testing: use test ad unit ID `ca-app-pub-3940256099942544/1033173712`
